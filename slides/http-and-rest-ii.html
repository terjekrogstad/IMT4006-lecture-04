HTTP, URIs and REST II

The anatomy of a REST request

  • HTTP Verb: GET, POST, PUT or DELETE
  • Protocol: HTTP or HTTPS
  • Domain
  • Resource collection name
  • Resource identifier
  • GET protocol://domain.com/resources/identifier 
    Put together (First example is preferrable):
    
    GET http://carservice.com/cars/TF90000

    GET http://carservice.com/cars.cfm?reg=TF90000
