REST And SOAP

SOAP 1.1 and SOAP 1.2 supports a HTTP binding, which uses HTTP GET

HTTP GET does not allow a request body so there can't be any SOAP content in the request

    
    GET /www.socweather.com/temp?city=Honolulu&when=now HTTP/1.1
    Host: www.socweather.com
    Accept: text/html, application/soap+xml
